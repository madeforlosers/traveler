# traveler
a language i decided to make a while ago (5/9/25) and rediscovered.

has 18 functions as lowercase letters, technically 43 because if a function is uppercase it prints as well as does its designed action (if there is no function assigned to the letter, it just prints)

one of its main gimmicks is the use of Ans and BackupAns, a temp variable that every function uses

all functions only need at most 2 arguments, the first one is the Ans variable and the second one can be defined with `z`.

you can manually set 1-9 to ans by using it in place of a function . but you can't do anything higher than 10 without math example: 

`53zM` is 15 (5 * 3). `53zA` and `5z3A` is 8 (5 + 3). `10z2S` is 2 (2 - 0, not 2 - 10) 

the Ans variable will ALWAYS be the first argument. no matter what.
